# Superstore Sales Analytics & Forecast Dashboard

# Project Overview
This project presents a Sales Analytics & Forecast Dashboard built using Power BI to analyze historical sales performance and predict future trends.
The dashboard provides actionable insights into:

- Sales performance
- Profit trends
- Customer segments
- Shipping and payment behavior
- Regional and state-level analysis
- Short-term sales forecasting (15 days)

It enables data-driven decision-making for business growth and operational efficiency.

---

# Features 🎯
1. KPI Cards
- Total Sales: 1.6M
- Total Quantity Sold: 22K
- Total Profit: 175K
- Average Delivery Days: 4

2. Sales Analysis
- Sales by Segment (Consumer, Corporate, Home Office)
- Sales by Category (Office Supplies, Technology, Furniture)
- Sales by Sub-category (Phones, Chairs, Binders, etc.)
- Sales by Ship Mode (Standard, Second Class, First Class, Same Day)
- Sales by Payment Mode (COD, Online, Cards)

3. Time-Based Analysis
- Monthly Sales Trend (Year-wise comparison)
- Monthly Profit Trend (Year-wise comparison)

4. Geographic Insights
- Sales & Profit by State (Map Visualization)
- Top-performing states (California, New York, Texas, etc.)

5. Forecasting
- 15-day Sales Forecast using historical trends
- Confidence interval visualization for predictions

# Key Insights 📈
- Consumer segment dominates sales (~48%)
- Office Supplies category leads revenue (~0.64M)
- Phones and Chairs are top-selling sub-categories
- Standard Class shipping contributes the highest sales (~0.91M)
- COD is the most preferred payment mode (~43%)
- Sales show a strong upward trend in the last quarter (Oct–Dec)
- Profit peaks in November and December
- California generates the highest sales (~0.34M)

# Forecast Insights 🚀
- Sales are expected to remain stable with slight growth
- Forecast confidence interval indicates moderate variability
- Useful for short-term inventory and demand planning
